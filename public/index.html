Route → GPX

Paste a Google Maps link, get a GPX file for your Wahoo.

Google API Key
Needed to fetch route data from Google. Your key stays on your phone — nothing is sent to any server except Google's API.

Quick setup: Go to Google Cloud Console, create a project, enable the Directions API and Maps JavaScript API, create an API key, and paste it above.
Google Maps Route
Open Google Maps → get directions → tap Share → Copy link → paste here. Use the full URL (shortened maps.app.goo.gl links won't work — open them in Chrome first and copy from the address bar).
